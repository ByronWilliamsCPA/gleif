## [0.1.0] - 2026-04-24

### Added

- Initial release: GLEIF golden copy CLI with DuckDB backend
- Commands: download, load, refresh, lei, name, status
- LEI relationship hierarchy queries (parents, children, siblings)
- Reporting exception support
- ISIN lookup via GLEIF REST API
- Three GLEIF datasets: LEI records, relationships, reporting exceptions
- Rich terminal output with formatted tables and trees
